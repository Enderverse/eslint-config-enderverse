## Install

npm
```sh
npm install Enderverse/eslint-config-enderverse --save-dev
```

yarn
```sh
yarn add Enderverse/eslint-config-enderverse -D
```


## Usage

Add the following to your `.eslintrc` or `.eslintrc.json`:
```json
"extends": "enderverse"
```
