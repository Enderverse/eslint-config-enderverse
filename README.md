## Install

npm
```sh
npm install enderverse/eslint-config --save-dev
```

yarn
```sh
yarn add enderverse/eslint-config -D
```


## Usage

Add the following to your `.eslintrc` or `.eslintrc.json`:
```json
"extends": "@enderverse"
```
